# GDDV402-Assignment2
Assignment #1: In-App Purchasing
Worth 15%


Objective:

    Build and deploy an APK with IAP


Instructions for Unity Project

    Install the necessary dependencies and create a new Android project in Unity. [5 points]
    Define your app identification and app icon in Player Settings and enforce Landscape orientation. [5 points]
    Implement a simple UI to display a total of green gems as a currency. [2 points]
    Design and integrate a simple menu to enable users to "purchase" one of three in-app purchases: 2 consumable or 1 non-consumable.
    Add a button to "Buy 1 Gem", which adds 1 green gem to the total. [2 points]
    Add a button to "Buy 10 Gem", which adds 10 green gems to the total. [2 points]
    Add a button to "Upgrade Gems" to permanently upgrade gems from green to red. [2 points]
    Implement a Gem Manager that decides what gem type to display on the UI. [2 points]


Submission Requirements

Include your completed worksheet inside your ZIP submission file. The hierarchy of the ZIP's contents should be:

    Lastname-Firstname-A1.zip
    Root folder - [Last name]-[First name]-A1
    Unity project folder
    Assets
    Packages
    ProjectSettings
    APK file
